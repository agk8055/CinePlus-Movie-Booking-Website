CinePlus API Documentation

Base URL: /api/v1

Authentication

POST /auth/send-otp
Send OTP for signup verification
POST /auth/complete-signup
Verify OTP and complete user registration
POST /auth/login
User login
GET /auth/me
Get logged in user's profile (Protected)

Movies

GET /movies
Get all movies
GET /movies/:id
Get movie by ID

Theaters

GET /theaters
Get all theaters
GET /theaters/:id
Get theater by ID

Showtimes

GET /showtimes
Get all showtimes
GET /showtimes/:id
Get showtime by ID

Bookings

POST /bookings
Create a new booking
GET /bookings/:id
Get booking by ID

Seats

GET /seats
Get all seats
GET /seats/:id
Get seat by ID

Cities

GET /cities
Get all cities

Payments

POST /payments
Create a new payment

Users

GET /users
Get all users (Admin only)
GET /users/:id
Get user by ID

Theater Dashboard

GET /theater/dashboard
Get theater dashboard data (Theater Admin only)
